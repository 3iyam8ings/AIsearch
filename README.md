## 🛠️ Tech Stack

**Frontend:**

- [React 19] + [TypeScript]
- [Vite] (Build Tool)
- [React Router] (Navigation)
- Vanilla CSS (Custom Glassmorphism & Animations)

**Backend:**

- [Bun] (Runtime)
- [Express] (Server)
- [Prisma] (ORM)
- [PostgreSQL] (Database)
- [Supabase] (Auth / JWT Validation)
- [Tavily Search API]
